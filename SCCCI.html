SCCCI Chatbot Admin System

AI-powered Chatbot Management Platform for SCCCI
About the Project:
SCCCI Chatbot Admin System is a comprehensive platform for managing chatbot interactions, analytics, and user queries for the Singapore Chinese Chamber of Commerce & Industry. The system features a responsive admin dashboard, real-time data visualization, and secure authentication for administrators.
Key Features:
  • Built with React and JavaScript for real-time chatbot management
  • Backend services with Node.js and Express for robust server-side functionality
  • JWT-based authentication and role-based authorization
  • Data visualization and management capabilities
