SGHawkers Platform

A Modern Hawker Centre Management & Food Accessibility Solution
About the Project:
SGHawkers is a web platform designed to streamline hawker centre management and improve food accessibility for the community. The system features real-time stall selection, order tracking, and a user-friendly dashboard.
Key Features:
  • Stall selection and real-time order status
  • Customer can send special requests for the specific dish they order
  • Sending live notification after successful order
  • Sending review through feedback
